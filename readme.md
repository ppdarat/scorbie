# scorbie

Scorbie is a modern, lightweight badminton scoreboard application built with React, TypeScript, and Vite.

## features

- **score tracking**: simple and intuitive score tracking for two teams.
- **skunk rule**: instant win support if a team reaches 3-0.
- **deuce logic**: requires a 2-point lead when scores are tied at target-1.
- **serving logic**: indicates which side (left/right) should serve based on the current score.
- **customizable settings**: adjust target score, and toggle rules (skunk, deuce, serving logic) on or off.
- **responsive design**: works seamlessly on desktop and mobile devices.

## setup

1. install dependencies:
   ```bash
   npm install
   ```
2. start the development server:
   ```bash
   npm run dev
   ```
3. build for production:
   ```bash
   npm run build
   ```
